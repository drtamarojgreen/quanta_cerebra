# QuantaCerebra User Manual

## Overview
QuantaCerebra is a brain activity simulation and visualization tool.

## Configuration
Settings can be adjusted in `config.json` or `config.ini`.

## Interactive Controls
- `P`: Pause/Resume
- `F`: Increase speed
- `S`: Decrease speed
- `Z`: Zoom in
- `X`: Zoom out
- `H/J/K/L`: Pan view

## Layout Modes
- `list`: Standard region list
- `grid`: 2D spatial grid
- `3d`: Isometric 3D projection
