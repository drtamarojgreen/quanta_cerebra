# Changelog

## [2.0.0] - 2023-11-20
### Added
- **Modular Architecture**: Reorganized codebase into domain-specific modules (`core`, `ai`, `analytics`, `cloud`, `io`, `ui`, `visualization`).
- **Advanced AI Integration**:
  - NeRF-inspired 3D Cortex Raymarching.
  - Video Diffusion denoising simulation.
  - Neural Cellular Automata for region interaction.
  - LLM Hub for pattern generation.
- **Real-time Analytics**:
  - FFT and PCA implementations for frequency and component analysis.
  - Granger Causality and Entropy calculations.
  - Connectivity Matrix generation.
- **Cloud Ecosystem**:
  - Simulated AWS S3, Lambda, and Kinesis integration.
  - HashiCorp Vault authentication simulation.
- **Expanded IO**:
  - Parsers for XML, YAML, and CSV brain activity data.
  - Exporters for MIDI, PNG, GIF, BMP, and SVG.
- **Enhanced Visualization**:
  - 3D Greenhouse View with pseudo-projection.
  - 2D Grid/Matrix layout.
  - Particle system rendering.
  - 256-color ANSI theme support.
- **Robust Testing**:
  - Comprehensive 4-tier testing (Unit, BDD, E2E, SDD).

## [1.0.0] - 2023-05-15
### Added
- Initial release with basic ASCII visualization.
